クイズを遊ぼう！の使い方

用意されたクイズを選んで遊べるクイズプレイヤーです。ヒント機能や進捗管理に対応しており、途中で中断しても続きから再開できます。

基本的な使い方

  1. クイズ一覧から遊びたいクイズを選択します。
  2. 問題が表示されたら、回答を入力または選択してください。
  3. わからない場合はヒント表示機能を活用できます。
  4. 回答後、正誤の結果が表示されます。
  5. すべての問題を終えると完了となります。途中で中断した場合は、次回アクセス時に続きから再開できます。
  6. 完了後はリトライ機能でもう一度挑戦できます。

主な機能

  • クイズ一覧からの選択
  • 段階的な進行（問題表示 → 回答 → 結果）
  • ヒント表示機能
  • 中断・再開対応
  • リトライ機能
  • 進捗管理（未開始 / 進行中 / 完了）

入力と出力

入力: クイズの選択、各問題への回答

出力: 正誤判定の結果、進捗状況の表示

補足情報

  • 進捗状況はブラウザに保存されます。ブラウザのデータを消去すると進捗もリセットされます。
  • 進捗は「未開始」「進行中」「完了」の3段階で管理されます。
  • ヒントは問題ごとに用意されていますが、すべての問題にヒントがあるとは限りません。
  • リトライ時は進捗がリセットされ、最初からやり直しになります。
