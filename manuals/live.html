AIと一緒に作ったWebツールを公開しています。

ライブ情報まとめツールの使い方

楽曲のライブ情報を管理するためのツールです。楽曲ごとにシーン情報やパート分けを登録し、キャラクター配置のシミュレーションや各種形式での出力ができます。

基本的な使い方

  1. 楽曲の基本情報を設定します。タイトル、歌唱可能キャラクター、配置人数、歌唱人数を入力します。
  2. シーン情報を登録していきます。各シーンの再生時間、歌詞、パート分け、備考などを入力します。
  3. シミュレーション機能を使って、キャラクターの配置を可視化し確認します。
  4. データの入力が完了したら、用途に応じてJSON・TXT・HTMLのいずれかの形式で出力します。

主な機能

  • 楽曲基本情報の設定（タイトル、歌唱可能キャラ、配置人数、歌唱人数）
  • シーン情報の登録（再生時間、歌詞、パート分け、備考）
  • キャラクター配置のシミュレーション・可視化機能
  • JSON / TXT / HTML の3形式でのデータ出力
  • 1曲単位でのデータ管理

入力と出力

入力: 楽曲のタイトル・歌唱キャラクター情報、各シーンの再生時間・歌詞・パート分け・備考をフォームから入力します。

出力: 入力したライブ情報を3つの形式で出力できます。JSON形式はデータの保存・再読み込み用、TXT形式はテキストベースの共有用、HTML形式は閲覧用ページとして利用できます。

補足情報

  • データは1曲単位で管理されます。複数曲を扱う場合は、それぞれ個別にデータを保存してください。
  • スマートフォンでも使用できますが、画面サイズに最適化されていないため、PCでの利用を推奨します。
  • シミュレーション機能では配置の確認ができますが、実際のライブ演出とは異なる場合があります。
