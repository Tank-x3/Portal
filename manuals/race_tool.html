オリウマレース開催支援ツールの使い方

ダイスロールを使ったレースシミュレーションツールです。出走者の登録からレース進行、結果の保存まで、レース開催に必要な一連の流れをサポートします。

基本的な使い方

  1. レース設定を行います。中盤ダイス回数（0〜4回）とフルゲート人数（1〜100人）を指定してください。
  2. 出走者を登録します。名前、脚質（大逃げ / 逃げ / 先行 / 差し / 追込の5種類）、固有スキル（安定型 / ギャンブル型）を設定します。
  3. 全出走者の登録が完了したら、レースを開始します。
  4. レースはフェーズ制（Start → Pace → Mid → End）で自動進行します。
  5. レース終了後、順位表が表示されます。上位3位はハイライト表示されます。
  6. 順位表はテキストコピーまたはPNG画像として保存できます。

主な機能

  • ダイスロールベースのレースシミュレーション
  • レース設定のカスタマイズ（中盤ダイス回数: 0〜4回、フルゲート人数: 1〜100人）
  • 5種類の脚質設定（大逃げ / 逃げ / 先行 / 差し / 追込）
  • 2種類の固有スキル（安定型: 5+1d10 / ギャンブル型: 1d20）
  • フェーズ制レース進行（Start → Pace → Mid → End）
  • 順位表のテキストコピー機能
  • 順位表のPNG画像保存機能
  • 上位3位のハイライト表示

入力と出力

入力: レース設定（中盤ダイス回数・フルゲート人数）、出走者情報（名前・脚質・固有スキル）

出力: レース結果の順位表（テキストまたはPNG画像）

補足情報

  • 脚質によってフェーズごとのダイス補正が異なります。レース展開に大きく影響するため、出走者の特徴に合わせて選択してください。
  • 安定型スキル（5+1d10）は最低保証があり安定した成績が期待できます。ギャンブル型スキル（1d20）は振れ幅が大きく、大勝ちも大負けもあり得ます。
  • フルゲート人数は最大100人まで設定可能です。
  • 中盤ダイス回数を増やすと、レース中盤の展開がより変動しやすくなります。
